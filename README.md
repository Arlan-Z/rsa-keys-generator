# RSA Key Generator (React)

A browser-based RSA key generator with one-line output, where line breaks are replaced with `\n`.

## Security Notice

Keys generated by this tool are for demo/testing purposes only and should not be used in production.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Generates RSA keys (2048 / 3072 / 4096) using Web Crypto API.
- Outputs `PUBLIC KEY` and `PRIVATE KEY` as one-line PEM with `\n` separators.
- Supports copying keys individually or both at once.
- Provides a preview of standard PEM with real line breaks.
